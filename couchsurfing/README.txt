MMP COUCHSURFING

Implementierter Workflow:

Rolle "Couch-Anbieter" (Muster-User mit vorerfasster Couch und Buchungsanfragen: Login mit Email 111@111.ch, pw 111)
- User registrieren registrieren.html (bestehender Code)
- Login login.html (bestehender Code)
- Button "Meine Coach":
    - Neue Couch erfassen/ändern/löschen
    - Couch Verfügbarkeitsstatus ändern
    - Buchungsanfragen von Interessenten auflisten
    - Buchung verwalten (Anfrage annehmen oder ablehnen) mit Klick auf den jeweilige Buchungsanfrage-Link
    
Rolle "Gast"
- User registrieren registrieren.html (bestehender Code)
- Login login.html (bestehender Code)
- Couch suchen (Hauptseite)
    - Couch Angebote auflisten
    - Couch Angebote filtern nach Standort Bern/Zürich/Chur
    - Bei buchbaren Angeboten (Verfügbarkeitsstatus = Verfügbar) wird ein "Jetzt Buchung anfragen" Link angezeigt
    - Buchung anfragen inkl. Eingabe eines Bewerbungsgrunds ("Motivation")
    - Für die eigene Couch kann ein User keine Anfrage stellen

Jeder Benutzer kann beide Rollen "Anbieter" und "Gast" gleichzeitig einnehmen.
